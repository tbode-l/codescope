# CodeScope — GitHub 代码增强工具自动化竞品分析系统

> 课设产品设计 | 2026-06-06

## 做什么

一个纯 Python 命令行工具。输入关键词，自动从 GitHub 搜索同类代码增强工具，分析特征，评分排名，输出可视化报告。

## 怎么跑

```
python engine.py
```

四步一气呵成：

| 步骤 | 模块 | 做什么 |
|:--:|------|------|
| 1 | `tracker.py`（已有） | GitHub API 搜索 + 下载 README |
| 2 | `analyzer.py`（新建） | 从 README 提特征：技术栈、产品形态、安装复杂度 |
| 3 | `scorer.py`（新建） | 五维评分：活跃度、技术先进性、文档完整度、接入门槛、生态开放性 |
| 4 | `visualizer.py`（新建） | 输出雷达图 + 散点图 + 柱状图 |

## 产出

- `scores.json` — 所有工具排序评分
- 4 张 Matplotlib 图表
- 终端排名报告
